Profil Singkat

Seorang pengembang web yang berfokus pada pembuatan aplikasi berbasis PHP native dan MySQL. Tahapan Pengembangan

Fase 1 — Perencanaan
Analisis Kebutuhan & Desain Database
Menyusun ERD dengan 13 entitas, merancang relasi fisik 21 foreign key, mendesain struktur folder, dan membuat wireframe alur pembeli serta penjual.
Fase 2 — Fondasi
Setup Database & Autentikasi
Membuat database kantin, menjalankan skrip SQL iteratif (6 file), membangun sistem login/register pembeli dan penjual menggunakan session PHP.
Fase 3 — Inti Aplikasi
Keranjang, Checkout & Pembayaran
Mengimplementasikan keranjang belanja berbasis session, logika checkout multi-item dengan pemisahan per penjual, mekanisme upload bukti pembayaran, dan opsi Bayar di Kedai.
Fase 4 — Dashboard Penjual
CRUD Kedai/Menu & Kelola Pesanan
Membangun dashboard lengkap penjual: CRUD kedai dan menu, tabel pesanan masuk dengan kolom bukti bayar, pembaruan status pembayaran dan pengiriman.
Fase 5 — Penyempurnaan
Notifikasi, Profil & Integrasi
Membangun sistem notifikasi multi-tipe (pesanan, pembayaran, pengiriman) dengan tabel penghubung, halaman profil, serta integrasi OAuth Google, WhatsApp, dan Telegram.
